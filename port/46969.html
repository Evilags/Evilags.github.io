Butterfly主题优化

发表于2022-05-23|更新于2022-07-09|Butterfly主题
|阅读量:

1.在bolg根目录打开git bash here

2.输入 hexo new page link 此文件会在”E:\Blog\source\link\index.md”

3.在”Blog\source_data”文件夹下创建一个link文件夹并把文件尾缀设置为.yml —-“Blog\source_data\link.yml”

4.在link.yml文档里输入以下代码

  - class_name: 我的网站 🎉🎉   #主页标题栏名称
    class_desc: 成年人当然全都要。  #标题栏简介
        link_list:
      - name: 刘公子的博客    #名称
        link: https://intchiki.github.io/   #链接地址
        avatar: https://intchiki.github.io/touxiang.png   #头像链接
        descr: 本站 Butterfly 版   #名称下简介
      - name: guole's Blog
        link: https://blog.guole.fun/
        avatar: https://cdn.guole.fun/logo/blog.guole.fun.png
        descr: 本站 Hugo 版
      - name: 音乐小站
        link: https://music.guole.fun/
        avatar: https://cdn.guole.fun/logo/music.png
        descr: 一个小想法，造就了第一个小站
